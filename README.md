## 📌 About The Project

This project is a fully responsive landing page inspired by the design of **Radisson Blu** hotel branding.

The goal of this project was to practice building structured layouts using HTML and CSS, focusing on clean design, responsive behavior, and proper spacing across different screen sizes.

The layout adapts smoothly to mobile, tablet, and desktop devices.

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox
* Media Queries
* Responsive Web Design principles

## 📱 Features

* Fully responsive design
* Mobile-friendly layout
* Structured and semantic HTML
* Clean CSS organization
* Modern hotel-style UI layout
* Responsive navigation
